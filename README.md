### Keybinds
| | |
| :---: | :---: |
|**Arrow Keys**| Hitting notes |
| | |
|**Escape**| Pause menu |
|**1**| Show statistics |
|**2**| Cheat menu |
|**3**| Show timer |
|**4**| Collision counter |
